## Styled commit

#### 1. Install [git-cz](https://github.com/streamich/git-cz) global

```shell
npm install -g git-cz
```

#### 2. Commit files change

```shell
git add .
git cz
```

### -- Custom config

Edit `changelog.config.js` file in root folder
